<p align="center">
  <b>SkyMC – Addons & Themes Manager for Pterodactyl</b><br>
  Modern • Lightweight • Secure • Fully Customizable
</p>
SkyMC is a modern and lightweight Addons & Themes Manager built specifically for the Pterodactyl Panel. It provides a simple yet powerful system to install, manage, enable, and disable custom addons and themes directly from the admin dashboard. With automatic CSS and JavaScript injection, secure filesystem access, and full Laravel integration, SkyMC turns your panel into a fully customizable platform. Designed for developers, hosting providers, and anyone who wants complete visual and functional control over their Pterodactyl experience.
